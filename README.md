# portfolio
## About project
- Portfolio project is to organize our pictures or events and to display them.
## Requirements
- Knowledge on HTML
- Knowledge on CSS
- Updated web browser
## Description
- If we want to organise our content in our required manner the div tag in html plays a key role.
- In my portfolio project, I have used div tags, margin and padding tags to get my required picture organization.
- A separate css file is linked to the HTML file which styles the objects in HTML code.
## Running the project
- To run our HTML file, go to the file and simply run it in a updated web browser.
## Features of my project
- It displays some text regarding the corresponding image.
- By clicking on the link provided below every image, it will redirects to the wikipedia.
